Documentation

AdminKIT is a fully responsive html template, perfect for anyone looking to start a backend system with a modern interface. No more separate websites for mobile, as this theme will adapt seamlessly to any device screen and on top of that it's HD / retina ready.

Trough this documentation we have laid out information about using AdminKIT and in-depth customizing of elements in the item.
